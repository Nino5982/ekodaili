рЃЏрЃћрЃюрЃўрЃБрЃА рЃцрЃЮрЃюрЃў (Sidebar): #2D3E27 Рђћ рЃЏрЃБрЃЦрЃў, "рЃбрЃДрЃўрЃАрЃцрЃћрЃарЃў" рЃЏрЃгрЃЋрЃљрЃюрЃћ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЎрЃљрЃарЃњ рЃЎрЃЮрЃюрЃбрЃарЃљрЃАрЃбрЃА рЃЦрЃЏрЃюрЃўрЃА рЃЌрЃћрЃЌрЃа рЃбрЃћрЃЦрЃАрЃбрЃЌрЃљрЃю.

рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃдрЃўрЃџрЃљрЃЎрЃў (CTA): #7BB354 Рђћ рЃюрЃљрЃЌрЃћрЃџрЃў рЃЏрЃгрЃЋрЃљрЃюрЃћ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃљрЃАрЃЮрЃфрЃўрЃарЃЊрЃћрЃЉрЃљ Greenway-рЃА рЃЉрЃарЃћрЃюрЃЊрЃЌрЃљрЃю.

рЃЏрЃћрЃюрЃўрЃБрЃА рЃњрЃљрЃЏрЃДрЃЮрЃцрЃў рЃ«рЃљрЃќрЃћрЃЉрЃў: #A3C644 Рђћ рЃБрЃцрЃарЃЮ рЃЏрЃЎрЃЋрЃћрЃЌрЃарЃў рЃЏрЃгрЃЋрЃљрЃюрЃћ рЃљрЃЦрЃфрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
