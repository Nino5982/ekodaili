# Greenway Georgia - Landing Page ­Ъї┐

рЃћрЃА рЃљрЃарЃўрЃА рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў Landing Page, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ Greenway-рЃА рЃърЃарЃЮрЃЊрЃБрЃЦрЃфрЃўрЃўрЃА рЃърЃЮрЃърЃБрЃџрЃљрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃцрЃЮрЃЎрЃБрЃАрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃљрЃдрЃљрЃџ рЃЎрЃЮрЃюрЃЋрЃћрЃарЃАрЃўрЃљрЃќрЃћ, рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃа рЃърЃўрЃарЃЋрЃћрЃџрЃЮрЃЉрЃљрЃќрЃћ (Mobile-First) рЃЊрЃљ рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃБрЃа рЃАрЃўрЃќрЃБрЃАрЃбрЃћрЃќрЃћ.

## ­Ъџђ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў рЃЊрЃљ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў

* **Responsive Design:** рЃАрЃљрЃўрЃбрЃў рЃАрЃарЃБрЃџрЃљрЃЊ рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃўрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃќрЃЮрЃЏрЃўрЃА рЃћрЃЎрЃарЃљрЃюрЃќрЃћ (Desktop, Tablet, Mobile).
* **Scroll Spy:** рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃЏрЃћрЃюрЃўрЃБ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃюрЃљрЃЌрЃћрЃЉрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃАрЃћрЃЦрЃфрЃўрЃљрЃА рЃАрЃЎрЃарЃЮрЃџрЃўрЃюрЃњрЃўрЃАрЃљрЃА.
* **Performance:** рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃў JS рЃАрЃгрЃарЃљрЃцрЃў рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА.

## ­ЪЊі рЃЏрЃљрЃарЃЎрЃћрЃбрЃўрЃюрЃњрЃў рЃЊрЃљ рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃљ

рЃАрЃљрЃўрЃбрЃў рЃАрЃарЃБрЃџрЃљрЃЊ рЃљрЃарЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў Meta-рЃА рЃЊрЃљ Google-рЃўрЃА рЃћрЃЎрЃЮрЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃЌрЃљрЃю:

* **Meta Pixel:** рЃЕрЃљрЃерЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃЌрЃљ рЃЦрЃфрЃћрЃЋрЃўрЃА рЃЊрЃљрЃАрЃљрЃбрЃарЃћрЃЎрЃљрЃЊ (ID: `1462012855339552`).
* **Event Tracking:** рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃўрЃњрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ `Lead` рЃЊрЃљ `Contact` рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃў рЃЎрЃљрЃбрЃљрЃџрЃЮрЃњрЃќрЃћ рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃўрЃАрЃљрЃА рЃЊрЃљ WhatsApp-рЃўрЃЌ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃўрЃАрЃљрЃА.
* **Google Analytics (GA4):** рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃЌрЃљ рЃюрЃљрЃЎрЃљрЃЊрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃўрЃАрЃЌрЃЋрЃўрЃА.
* **Open Graph (OG) Tags:** рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃўрЃарЃћрЃЉрЃБрЃџрЃў Social Preview Facebook, Instagram рЃЊрЃљ WhatsApp рЃњрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
